如果你是苹果电脑的用户，那么想如同PC一样寻找一款自己喜欢的、性能足够好的显卡则并不容易，不过好在音频系统方面我们可以有一些选择。虽然Macbook或iMac相对PC仍小众得多，但如何获得比集成声卡更好的音质当然需要选择一款外置声卡，在很多年前，更多的是来自专业品牌的音频设备，价格高昂。而大家熟悉的小夜曲USB声卡，目前也提供了Mac OS X下的驱动，而且可以支持高品质的ASY模式。我们今天在Mac OS X 10.7.2下对这款声卡进行测评。随后，我们还会对E-MU的0204 USB在Mac下的表现进行测评。
![TempoTec HiFier Serenade 节奏 小夜曲USB 声卡](https://images.soomal.cc/images/doc/20110927/00013817.webp)




![TempoTec HiFier Serenade 节奏 小夜曲USB 声卡](https://images.soomal.cc/images/doc/20110927/00013818.webp)




我们使用的Macbook比较老，它是一款2007年发布的使用酷睿2 T8100处理器的“小白”，不过操作系统方面，我们使用了最新的Mac OS X 10.7.2[Lion]，所以对于目前所有的Mac用户都有参考意义。初次使用小夜曲USB，和Windows下稍有不同，我们接下来进行简单介绍。
![TempoTec HiFier Serenade 节奏 小夜曲USB 声卡-Mac OS X下设置](https://images.soomal.cc/images/doc/20111128/00015150.webp)




![TempoTec HiFier Serenade 节奏 小夜曲USB 声卡-Mac OS X下设置](https://images.soomal.cc/images/doc/20111128/00015151.webp)




无论你将小夜曲USB前面板的ADA/ASY按钮放在什么位置，连接到Mac OS X后，我们都可以在“实用工具-音频MIDI设置”的项目中找到设置界面，并看到小夜曲USB的设备，但此时系统会显示它的名字为"Hifier Serenade USB ASY"。这并不能说明它已经工作于ASY状态，只是读取了设备的名称而已。我们必须安装节奏提供的驱动程序，并重启系统，并将声卡的开关放置ASY模式下，它才完全工作于此状态。

在Mac OS下，声卡的采样率和采样精度同样需要在如图的设置页面中进行手动的设置，并不可以像ASIO或Windows 7下的WASAPI模式下自动识别并指定设备的工作模式。手动调整采样率后，小夜曲USB的指示灯起到了作用，这时候可以立刻看到响应，即便没有任何曲目在播放也可以看到指示灯的切换。为了让播放器，如iTunes可以使用小夜曲USB声卡作为默认播放设备，我们需要在左下角进行相关设置。

客观测试
![E-MU 1616M PCIe 数字音频系统](https://images.soomal.cc/images/doc/20101204/00008507.webp)




我们使用iTunes播放器进行相关测试，在iTunes的偏好设置中，有“声音增强器”一项，使用时不推荐打开，它会对低频和高频的频响有很明显的改变。小夜曲USB在Windows下输出电平相对偏高，播放标准电平信号时，默认输出达到了-0.5dB左右，而在Mac下，小夜曲USB的输出为-2.5dB左右，我们在录音端对输入电平进行补偿至-1dB。我们对LineOut进行一组测试，测试录音设备仍为E-MU1616m。测试时，我们使用ASY模式，也推荐大家使用该模式。

从客观测试数据来看，Mac下面小夜曲USB的测试数据与Windows下没有什么区别，超过96dB的数值超过了理论最大值，属于测试误差，这与小夜曲USB输出电平偏高以及我们在录音端的电平补偿有关。在48kHz采样率测试下，没有出现SRC的问题。不过，我们也测试了Macbook[MB402]集成声卡，发现它在44.1kHz播放时出现SRC的问题，只在高频时有一定影响。

另外，我们在这里也顺便测试了小夜曲USB用户反映的焦点问题――CPU占用率。如图所示在这台Macbook下，它的CPU占用率没有异常，主要的CPU占用率为“性能监视器”本身和iTunes，而它的CPU是Intel T8100[2.1GHz Core 2 Duo]。这里顺便谈一下小夜曲USB CPU占用率的问题，据官方解释，这是因为驱动程序在Windows 7下不断请求系统I/O中断所致，在Windows 7下，各个程序占用线程里的确看不到处理器的对应使用情况。一般2007年以后的主流配置，即酷睿2以后的处理器性能可以足够应付。AMD平台据说有更多不明的不稳定情况……

![TempoTec HiFier Serenade 节奏 小夜曲USB 声卡-Mac OS X Lion下CPU占用率](https://images.soomal.cc/images/doc/20111128/00015153.webp)




主观听感：由于小夜曲USB在Mac下的驱动程序指示的问题，我们并不能完全确定它的ASY工作是否与Windows下一样，毕竟在Windows下ADA模式同样可以得到很漂亮的客观测试数据，我们只能靠主观听音感受来再次确认小夜曲USB的输出品质。我们对比Windows下小夜曲USB的表现，可以确认小夜曲USB在Mac下的确工作于ASY状态，它的声音动态整体会稍好，高频部分更通透，细节更有小夜曲USB修饰的味道。而在ADA模式下，声音层次会稍差一些。

总体来说，小夜曲USB的Mac OS驱动可以很好的让硬件工作于Lion操作系统下，对于Mac用户小夜曲USB是一个高品质声音输出的方案之一。可惜的是，小夜曲USB的外形也许和大多数用户选择苹果产品的外观风格要求有不小的差距。
