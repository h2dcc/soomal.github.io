AirDots 3 是红米品牌推出的一款蓝牙真无线耳机，圈铁扬声器，售价199元人民币，SOOMAL也想看看各种品牌的入门级产品表现怎么样。

硬件设计
![红米 Redmi AirDots 3 蓝牙真无线耳机](https://images.soomal.cc/images/doc/20210512/00094187_01.webp)




![红米 Redmi AirDots 3 蓝牙真无线耳机](https://images.soomal.cc/images/doc/20210512/00094188_01.webp)




AirDots 3有白粉蓝三色可选，顶视造型就是一个跑道形，造型线条处理有柔有刚，整体简洁大方。电池盒壳体磨砂处理，手感比较细腻，也比较耐划，但抗油污稍差，用久了还是有点显脏，但比之前评测的网易云那款还是好得多。模具精度较好，在同价位中算得上优点，甚至比不少400价位段的产品更精致。盒盖铰链阻尼不是太均匀，盒盖也有松动感，但可以接受。
![红米 Redmi AirDots 3 蓝牙真无线耳机](https://images.soomal.cc/images/doc/20210512/00094191_01.webp)




![红米 Redmi AirDots 3 蓝牙真无线耳机](https://images.soomal.cc/images/doc/20210512/00094194_01.webp)




耳机造型外侧面高亮面装饰，其他部分与电池盒质感一致，造型风格与电池盒高度一致，其基础造型也是跑道形，同样也是简洁风。但耳机壳体造型不能只考虑简洁。AirDots 3耳机壳体内测并没有任何贴合耳部的设计，密封基本全依赖耳套实现。耳机导管较短，耳套的筒身也不长，这样会出现插入深度不太够的问题，影响佩戴的稳定性，我们找了多位人士佩戴该耳机，多数反馈“容易掉”或者“戴不牢”。”我们认为这个耳机壳体设计存在不足。
![红米 Redmi AirDots 3 蓝牙真无线耳机-结构示意图[来自官网]](https://images.soomal.cc/images/doc/20210521/00094388.webp)




AirDots 3采用动铁+动圈扬声器的组合，动铁负责高频，动圈负责低频。多单元的设计能够让频率响应变得更宽，但要最大限度的发挥多扬声器的优势，需要分频技术的支持，官网只提到“高中低频无缝衔接”，但并没有提及是如何分频的。

主控芯片采用高通QCC3040，支持蓝牙5.2，理论具有更低的功耗和连接稳定性。支持SBC、aptX、aptX Adaptive三种编码方案，不支持AAC，与iPhone以及华为系列机型搭配只能使用SBC。与大部分基于骁龙855及以上版本的处理器的手机相连，可以使用aptX Adaptive。aptX Adaptive可以用更小的码率能实现比aptX HD更好的编码性能，延时可以低至50-80毫秒[高通的官方数据]。

AirDots 3 单次续航7小时，连同电池盒补血续航，总续航时间30小时。实测中，将AirDots 3设置为中等音量，满电状态使用aptX Adaptive编解码，4小时持续播放后，剩余电量42%，与AirDots 3的单次续航时间能对的上。

被动降噪与语音降噪
2021年，SOOMAL尝试使用新的测试方法来改进部分测试项目，使测试结果更易于呈现。AirDots 3没有主动降噪功能，但它采用入耳式设计，可以实现被动降噪效果，在普通消费者那儿，也称为“隔音”。所以，也可以拿来按照新的测试方法来测一测。
![红米 Redmi AirDots 3 蓝牙真无线耳机-被动降噪测试](https://images.soomal.cc/images/doc/20210521/00094389.webp)




我们通过模拟不同环境的噪声进行评估，最适合呈现的噪声就是白噪声。我们制造了一个80dB左右的白噪声环境，上图最左侧是环境噪声直接录制，右侧的则是捕捉并测量AirDots 3被动降噪的的信号。
![SOOMAL 降噪数据表[2021年5月21日]](https://images.soomal.cc/images/doc/20210521/00094391.webp)




根据我们多年的经验，被动降噪耳机的噪声“重灾区”频段是20-1000Hz段，在4000+Hz段，被动降噪耳机通常比主动降噪耳机还干净。所以我们不能套用主动降噪耳机的评估方法来评估被动降噪性能。

非主动降噪的入耳式耳机通常可以获得比降噪耳机在被动降噪模式下更好的隔音效果，原因很简单，因为麦克风拾音孔更少。AirDots 3可以高效的隔离1000Hz以上的噪声，频率越高，隔离得越干净。得到的成绩非常优秀，和另外一个优秀者漫步者TWS1 Pro相比，AirDots 3在20-500Hz段表现更好，而20-1000Hz段评估稍稍逊色TWS1 Pro。但这里有个佩戴的问题不要忽视，在佩戴的稳定性方面，AirDots 3是不如TWS1 Pro的，AirDots 3更容易因为面部运动或者肢体运动导致漏音，从而影响被动降噪，甚至音质。
![红米 Redmi AirDots 3 蓝牙真无线耳机-语音降噪](https://images.soomal.cc/images/doc/20210521/00094390.webp)




这是AirDots 3的通话测试的录音。测试在40dB的室内、80dB的白噪声、75dB左右的人声嘈杂、车流噪声的模拟环境中进行，也进行了实景测试。AirDots 3 在正常佩戴下，在上述模拟的吵闹环境中的表现不佳，吞字现象较为严重，语义理解易产生歧义，在嘈杂的街头、餐厅、游乐场所中通话都可能出现听不太清楚的情况。为了减少理解歧义，AirDots 3的语音增益设置得比较大。

AirDots 3支持摘机通话，就是可以拿下一只耳机放在嘴边当麦克风用，这样可以在吵闹环境中提升语音清晰度。AirDots 3的这个功能需要改进，因为增益太大的问题，摘机通话的语音信号会出现削顶失真现象，最终导致语音通话质量仍然不佳。这属于电平管理策略问题，希望小米通过固件升级的方式进行修复，AirDots 3有佩戴感应，要做出正确判断并不难。

音质和总结
![红米 Redmi AirDots 3 蓝牙真无线耳机](https://images.soomal.cc/images/doc/20210512/00094191_01.webp)




![红米 Redmi AirDots 3 蓝牙真无线耳机-戴入耳模](https://images.soomal.cc/images/doc/20210512/00094196_01.webp)




 AirDots 3支持aptX Adaptive，我们手头基于高通骁龙888、870、865的手机，大部分都能完美配合。

![红米 Redmi AirDots 3 蓝牙真无线耳机-支持的编解码方案对比](https://images.soomal.cc/images/doc/20210522/00094399.webp)




这是SOOMAL测试的第二款支持aptX Adaptive的机型，所以我们还是再次通过可视化的方式来观察aptX Adaptive与其他编解码的不同。相比aptx，主信号上下侧的谐波消失了，这种棉花糖状的频谱特征是aptX以及aptX HD特有的，谐波消失，意味着信号品质有改善，在信号级这一环节就实现了音质提升。

高频表现：AirDots 3 使用了动铁高音单元，但和预期的还是有较大差别，199价位，不大可能上多好的扬声器。这枚动铁单元听感比较明亮，有点“铁味”，这个价位的大部分机型，高频都普遍偏暗。瞬态响应较快，动态偏小，密度较好，但声线谈不上细腻，高频的线条较粗，听感偏硬偏呆。对于入门级产品来说，这种高频并不算差，但也谈不上好。

中频表现：AirDots 3在中频上是花了点心思的，虽然没提什么技术手段来管理分频，但听得出还是下了功夫。它没有非常明显的女声齿音，声音的形态感尚可。AirDots 3的中频保持了较高的密度，偏高频段更为明显，而偏低频段，密度会下降，所以人声表现上女声整体好于男声，年轻女声会显得相对精神一点，但也会偏硬，整体来说，AirDots 3中频听感偏紧偏硬，其实可以调得更宽松点。

低频表现：AirDots 3的低频部分的表现不太好评价，它与佩戴情况相关。先说说佩戴严密的情况时的表现，这种状态下，低频量感较多，但偏肥，低频的细节、瞬态和动态都为量感做了牺牲，机内应该有处理。其他还会对中频偏低频段构成影响，此时的男低音可能会变得相对松垮。大部分时候，AirDots 3是出于佩戴漏音的状态下的，我们认为是耳机壳体设计的问题，小心佩戴好之后，可能说话就会导致耳机再次松动，这种状态下的低频量感会明显偏少，三频可能因此变得失衡。 

总结：AirDots 3 的声音表现并没有让人惊喜，其正常表现时的音质水平，也对得起价位。我们没听过AitDots旧款，因此也不知道有多大进步。从做工和用料来说，AirDots 3是有诚意的，不管如何，199元的产品做到这种精致度已经不错，至于动圈组合，也算是诚意的一部分，至少敢于尝试。最大的问题还是过于追求简洁，导致耳机佩戴存在问题，至少和我们的耳朵就不太“兼容”，在测评期间，我们总是不停的扶耳机，以保证佩戴严密性，因为它总是听着听着就开始漏音了，我们相信不是我们的耳朵长得太另类。因为佩戴问题也会导致对音质来自AirDots 3整体完全不太一样的评价，对这款有兴趣的，建议去实体店亲自肉身佩戴一下再做决定吧。
