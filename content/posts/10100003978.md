早晨起床是一件非常痛苦的事情，但工作要完成，生活要继续，这样懒散是肯定不行的。所以我们此时需要一个帮手，这就是智能手机，它在替代了闹钟、手表等传统工具之后成为新一代的“叫床利器”。不过问题是这个帮手有的时候并不管用，很多人在被唤醒之后会下意识的将手机闹钟关闭，继续睡觉，久而久之也就对此彻底免疫了。面对这样的情况，可能只有请出号称史上最烦的闹钟软件――“Sleep If  U Can”才能够解决了。



![Sleep If U Can](https://images.soomal.cc/images/doc/20130427/00030244.webp)



“Sleep If  U Can”意为“你睡个试试”或者“有本事你就睡”？它最早出现在Android平台上，目前也已经支持iOS系统。这款软件的工作原理是给手机闹钟加一个“锁”，让你不能够那么随便的关闭它。而在闹钟关闭之前，闹铃的音量一直处于最大状态[音量键失效]，让人烦不胜烦。当你实在是忍无可忍的时候，也就自然而然的醒来了。



下面，我们来看下“Sleep If  U Can”的使用。很简单，就是在添加闹铃时将它的关闭方法设置为拍照。此时，软件会提示你拍摄一张照片[或者从手机相册中读取]，这就是刚才提到的“锁”，而钥匙则是在闹铃启动后拍摄一张内容相同的照片[iOS版的功能更人性一些，拍照时会有原照片的轮廓提示]。当钥匙和锁匹配成功，你才可以关闭闹钟。这里建议大家最好拍摄家居中的一些物品比如冰箱、电视等，而随身物品如公交卡可能会丢失尽量不要使用。当然，你想自虐一下的话，也可以考虑拍摄夜景或者月亮之类……

此外，“Sleep If  U Can”的iOS版本还有另一个“锁”就是抖动[晃动设备]，即：设置的时候你可以选择晃动多少次才可以关闭闹钟。其中最自残的是50次，但对于一般人来说10-15次也就足够了。但要注意的是，摇晃N次设备这样的动作在家中也就罢了，但在室外很可能被人当做是抽风。我个人就曾经设置过3个闹钟，其中之一在乘坐公交时启动，于是吸引了周围各种异样的目光……

“Sleep If  U Can”这款闹钟软件虽然有时候很烦人，但实用性是比较强的，值得推荐给大家尤其是赖床专业户使用。当然，它能否正常工作也与用户的意志力成正比，因为你可以在铃声响起时选择强制关闭手机。所以，文章的最后想特别提醒大家一句：亲，上班迟到是要扣工资的！
