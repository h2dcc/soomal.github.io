关于手机的通话质量，有部分人认为“都差不多”，将通话中难以听清对方的声音全部归咎于信号问题，但实际情况如何呢？HTC One X是数码多手机通话质量测试的第二台智能手机，这也是我们日常使用较多的一部手机，对其通话质量我们并不满意，那么究竟是什么原因导致的？让我们通过测试来看一看。



![HTC One X [S720e]智能手机](https://images.soomal.cc/images/doc/20120515/00019631.webp)



外放音量测试



![HTC One X [S720e]智能手机](https://images.soomal.cc/images/doc/20120515/00019632.webp)



外放测试在环境噪声约为40dB的房间中进行，受测手机使用统一的铃声。声级计距手机一米，分别记录三组数据：手机正面向上平放、背面向上平放和装在相机包中的音量峰值。为缩小测试误差，以上测试重复多次，最终成绩以每组测试的中间值为准。



| 测试项目 |  |
| --- | --- |
| 正面平放, dB : | 69.7 |
| 反面平放, dB: | 74 |
| 放置于相机包内, dB: | 68.2 |



HTC One X的扬声器位于机身背面，虽然机背有一定弧度，但当手机正面向上平放在桌面上时扬声器仍然受到遮挡，因此音量比反面平放时下降了4.3dB，意味着音量降低接近40%。这样的设计不太合理，在实际使用中无论是放在桌面还是包中都会导致手机的铃声明显降低。







听筒测试



在听筒测试中，我们在信号稳定的环境中开启手机最大通话音量，拨打中国联通10010热线，使用专业声卡和高灵敏话筒，录下中国联通客服热线的固定语音，然后通过频谱和波形图进行分析。



![HTC One X [S720e]智能手机](https://images.soomal.cc/images/doc/20120515/00019634.webp)



![HTC One X 智能手机 - 通话波形记录图](https://images.soomal.cc/images/doc/20130503/00030486_01.webp)



![HTC One X 智能手机 - 通话记录光谱图](https://images.soomal.cc/images/doc/20130503/00030487_01.webp)



HTC One X听筒的峰值电平为-23.03dB，是我们近期测试的智能手机手机中最低的，比之前测试索尼Xperia Z[-14.36dB]小了8.67dB，这意味着HTC One X的听筒音量只有索尼Xperia Z的36.7%。通过声音光谱图，我们看到HTC One X的听筒信号频谱能量集中在4kHz以内，但出现许多超过4kHz的波峰会导致听感的劣化，对语音清晰度产生影响。分析频谱，会发现低频能量较弱，这对声音形态造成负面影响，实际听感是人声不够饱满而且人声“有点劈”了。HTC One X听筒音量小是一个比较严重的缺陷，加之语音回放质量也较差，导致实际使用中，经常难以听清楚对方说话。

降噪测试



降噪测试在一个安静[环境噪声约40dB]且信号良好的房间内进行，测试手机拨打电话至一台标准的电话设备，用专业声卡记录通话录音。测试用的通话内容来自一段固定的录音，其中包含完全相同的两个部分，测试中我们在第一个部分时制造一个强度约80dB的恒定噪声源，通过对比降噪前后通话记录的波形图和光谱图来分析测试手机的降噪表现。



![HTC One X 智能手机 - 降噪测试波形图](https://images.soomal.cc/images/doc/20130503/00030488_01.webp)



![HTC One X 智能手机 - 降噪测试光谱图](https://images.soomal.cc/images/doc/20130503/00030489_01.webp)



HTC One X的降噪采样时间比Xperia Z短，大约4秒钟。降噪采样速度应该是越快越好，这样才能应对突如其来的噪声。HTC One X的降噪效果尚可，能有效地抑制恒定噪声。在采样完成后，环境中的恒定噪声已不足以影响通话质量。从光谱图中可以看到，HTC One X的降噪算法对2kHz附近的频率存在负面影响，频谱上显示为“断开”的特征，说明有部分正常的语音信号被当做噪声滤除，这也导致了语音的清晰度受损，在小声说话时尤为明显。2kHz的部分信号损失会改变人声的声音特征，听感上的表现就是声音变得有气无力，不够饱满。HTC One X的语音质量问题用户本人可能不容易发现，而通话的另一方则会感觉到人声听起来单薄且有些口齿不清。这些问题反映出HTC One X的降噪算法还存在改进空间。







总结



![HTC One X [S720e]智能手机](https://images.soomal.cc/images/doc/20120515/00019630.webp)



HTC One X的通话质量是我们近期测试的手机中最差的，听筒音量过低、语音回放和通话质量较差等问题都会对通话体验造成双向影响。从使用体验来看就是通话双方经常需要重复叙述，并且人声显得有气无力。这与测试数据所反映的情况一致。显然，HTC One X的通话品质与用户的正常使用需求有较明显的差距。通过HTC One X与索尼Xperia Z的测试对比，可见目前智能手机的语音通话质量并不是如部分人所想的“都差不多”，不同的手机不仅听筒和语音的表现差距较大，降噪算法的效果也有不同。
