title: Technica 铁三角 AT-HA20 耳机放大器测评报告 不推荐评级
---

![Audio-Technica 铁三角 AT-HA20 耳机放大器](https://images.soomal.cc/doc/20101211/00008640.webp)



AT-HA20是铁三角旗下的一款入门级的耳机放大器，现售价在800元左右，由于是知名品牌，价格也在大部分消费者可接受范围内，加上日系产品一贯的精致做工等因素，这款产品有着不错的关注度。为了行文方便，下文中将其简称为HA20。文章的开始，特别鸣谢网友 守宫，是他提供的样品。

硬件与设计
![Audio-Technica 铁三角 AT-HA20 耳机放大器-与iPod Touch2大小对比](https://images.soomal.cc/doc/20101211/00008642.webp)




HA20体积不大。采用铝制机箱，机箱顶部安装了一块透明的亚克力板，通电后，机箱内的蓝色发光二极管会照亮内部，光线柔和，远看效果不错，但近看很容易看到那颗几乎不加隐藏而又体型硕大的发光二极管，配合不怎么好看的电路板，是真正的只可远观级的灯光设计。

![Audio-Technica 铁三角 AT-HA20 耳机放大器-前面板](https://images.soomal.cc/doc/20101211/00008643.webp)




前面板上有3.5mm和6.25mm两个耳机输出插座，以及电源开关和音量旋钮。两组耳机输出可以同时使用。

![Audio-Technica 铁三角 AT-HA20 耳机放大器-后面板](https://images.soomal.cc/doc/20101211/00008644.webp)




后面板上两组RCA，一组为输入，一组输出，HA20还能作为前级放大器使用。右侧插座是直流电输入，电压为12V。

![Audio-Technica 铁三角 AT-HA20 耳机放大器-功放板](https://images.soomal.cc/doc/20101211/00008646.webp)




拆开HA20，会发现与其他耳放不同的地方。PCB白色着色，配以浅棕色的丝印文字和符号，看上去相当精致。

![Audio-Technica 铁三角 AT-HA20 耳机放大器-UTC TEA2025](https://images.soomal.cc/doc/20101211/00008651.webp)




与我们之前测评过的几款耳放不同，HA20没有使用到运放或者三极管，它采用的集成电路方案，芯片为UTC出品的TEA2025。

TEA2025是一颗小功率的放大器芯片，这个型号年代久远，很多年前就被大量适用于便携磁带机或者收音机当中，在一些ISA老声卡上也能见到。但作为耳机放大器的驱动芯片使用，却并不多见。

这颗芯片输出功率并不大，它的工作电压很宽，最低可达3V，因此它的功率参数有很多版本，有2X2.3W，这应该是极限功率，也有2X1.3W的，这应该是其典型应用环境下的功率参数。这些参数的测量条件并不明，基本上应该都是按照负载4欧或者8欧，测试频率1kHz，总谐波失真上限10%的条件下测得。如果要用于耳机放大器，其可用功率要比这些功率低得多。

铁三角HA20官方标称的最大输出功率参数为：100mW + 100mW (32Ω、T.H.D. 5%)，并给出了在32Ω 20mW输出时的总谐波失真小于0.08%的标准。这一方面反应了它基本只适合中低阻耳机，而且应付大尺寸中低阻耳机还有很大可能表现不佳。

从硬件设计上来看，HA20不大可能具有良好的输出素质，但也不排除好的设计让平凡的元器件释放出能量的可能性。

客观测试与分析
![EMU Emulator X Studio 桌面采样系统](https://images.soomal.cc/doc/20100604/00005777.webp)




关于耳机放大器的客观测试，我们还是选用专业级声卡录入的形式，结合常见的分析方式，提供直观的测量结果。警告：民用声卡用户请勿尝试这种方式测试，我们无法确保信号过载时不会对声卡产生伤害。这种方式测试有其局限性，它与实际使用有一定差异，实际使用时会有不同的负载。但它反应的数据依然有参考性，尤其对左右声道的平衡性、分离度等指标的分析，能更加直观。 

![Audio-Technica 铁三角 AT-HA20 耳机放大器-RMAA 频响曲线](https://images.soomal.cc/doc/20101220/00008786.webp)





| 测试项目 | 结果 |
| --- | --- |
| 噪声水平, dB (A): | -91.7 |
| 动态范围, dB (A): | 91.6 |
| 总谐波失真, %: | 0.041 |
| 互调失真, %: | 0.040 |
| 立体声分离度, dB: | -75.9 |



HA20具有一条平直的频响曲线，但测量发现，左右声道差异0.5dB，虽然这种差异未必能被人耳感知，但这个差异已经是我们已测试耳放中最大的了，通常情况下，我们会将0.2dB左右的差异视为测试误差。

对比之前其他耳放的测试结果，会发现HA20的互调失真和总谐波失真表现也欠佳，它高于之前测试的HA-1、熊猫、莱曼、SOLO 10-20倍左右。

![铁三角 AT-HA20 耳机放大器-分离度-左声道](https://images.soomal.cc/doc/20101220/00008789.webp)




![铁三角 AT-HA20 耳机放大器-分离度-左声道-频率分析](https://images.soomal.cc/doc/20101220/00008790.webp)




![铁三角 AT-HA20 耳机放大器-分离度-右声道](https://images.soomal.cc/doc/20101220/00008791.webp)




![铁三角 AT-HA20 耳机放大器-分离度-右声道-频率分析](https://images.soomal.cc/doc/20101220/00008792.webp)




我们使用专业声卡捕捉信号并进行观察和分析，得到的结果如上图。串音比较严重，而且测试发现，左右声道的串音程度和曲线特征也不相同，起初我们认为是测试出现问题，但反复测试，才确认结果，就是HA20的左右声道一致性确实存在问题。另外，测试发现，其背景并不干净，存在噪声。

![Audio-Technica 铁三角 AT-HA20 耳机放大器-噪声](https://images.soomal.cc/doc/20101220/00008788.webp)




因此我们单独测试其无信号状态下的噪声，结果发现噪声较大，频谱中呈现很淡的蓝色，电平读数约为-76dB，在2kHz的位置上，有一条较为明显的噪声带。-76dB，与RMAA结果有明显差异，我们认为这个结果更加真实可信。


主观听感测试
为了与HA20更好的搭配，我们尽量选择低阻抗耳机包括铁三角ATH-AD1000、飞利浦 SHP9000、AKG K701、漫步者H800，和一款小耳塞创新Aurvana Air作为主要回放参考耳机，使用森海塞尔HD650作为高阻抗耳机参考。音源方面，我们使用乐之邦MD11解码器和乐之邦Monitor 02 US作为音源输出至耳机放大器，并使用它们的耳机输出作为对比参考。

![Audio-Technica 铁三角 ATH-AD1000 头戴式耳机](https://images.soomal.cc/doc/20101108/00008052.webp)




![乐之邦 Musiland Monitor 02 US USB 聆听二号声卡](https://images.soomal.cc/doc/20090618/00002141.webp)




![Musiland 乐之邦 MD11 外置立体声解码器](https://images.soomal.cc/doc/20100725/00006495.webp)




![Sennheiser 森海塞尔 HD650 头戴式耳机](https://images.soomal.cc/doc/20100703/00006217.webp)




![AKG K701 头戴式耳机](https://images.soomal.cc/doc/20100722/00006453.webp)




![Philips飞利浦 SHP9000 头戴式耳机](https://images.soomal.cc/doc/20100908/00007107.webp)




高频表现：以铁三角AD1000、飞利浦SHP9000、K701在HA20上表现来看，高频明显不能令人满意，若要保证这几款耳机高频能够较稳定发挥，不出现明显尖锐、干瘪的声音，几乎HA20只能工作在主观听感偏小的音量下（以MD11 LineOut最大电平输出，HA20设置在9-10点左右。）而即便中小音量时，高频明显偏硬，声音发干。与Monitor 02 US两个耳机输出任何一个相比，HA20表现全方面落后。Monitor 02 US，驱动以上三款耳机，高频明显细节要丰富得多，瞬态明显好得多，声音舒展饱满。而HA20，则显得声音发干，而其驱动力也相对一般，稍大动态时声音层次已经很差。

而使用HD650时，单从HA20推动的音量来看，还是足够大，倒不会遇到在板载声卡上输出电平都很弱的情况，但出现的问题与驱动低阻耳机基本一样。

而在使用振膜尺寸较小的漫步者H800时，HA20的表现要比以上几款耳机好不少，虽然在解析力上仍要弱于驱动它的音源，但即便在较大动态、较大音量时声音仍保持的较稳定，高频没有明显的毛糙和干瘪的感觉，但驱动力方面还要较弱于Monitor 02 US，动态稍显不足。


中频表现：HA20的中频控制力相对高频来说更差一些，人声在中小音量时已经出现一些声音发毛粗糙的感觉，音源有如此表现实在是不可以忍受的。与Monitor 02 US耳机输出相比，无论解析力、结像、声音层次还是驱动力来说，HA20都有十分明显差距。而至于大动态的交响乐，HA20表现的更为混乱。

同样，在使用漫步者H800时，HA20的中频控制相对要比以上三款较大尺寸的低阻耳机更好，声音层次较好，但驱动力一般，H800的中频显得有些偏闷，这倒于很多普通声卡直推H800时的感觉非常相似。

低频表现：如果使用Monitor 02 US作为音源，那么HA20在低频上只会带来更差的改变，在控制力和大动态表现上并不比原声卡耳机输出好。而在声音细节方面，明显会变得更差。对于几款低阻抗大尺寸的头戴式耳机来说，均存在几乎一样的毛病。而对于高阻抗HD650来说，驱动力也明显要比Monitor 02 US耳机输出更差，低频量感明显不足，更不要谈什么细节。

在驱动漫步者H800时，HA20的低频控制的较好，H800并没有出现在普通板载声卡上感觉低频过多声音浑浊的情况，H800的低频明显紧凑了许多，量感相对Monitor 02 US输出还要更少，动态表现明显不如Monitor 02 US，但相对驱动K701等耳机来说，HA20配合H800表现已经令人欣慰……

另外有趣的发现，在创新Aurvana Air小耳塞上，HA20的控制力似乎好于Monitor 02 US一些，但解析力仍有较大损失，声音发干，细节远不如Monitor 02 US。

总结
铁三角HA20耳放的驱动力非常有限，即便是配合低阻耳机，也只能局限于中小尺寸的耳机。它的作用几乎仅仅是在与普通板载声卡搭配时，让它驱动耳机的音量显得更大，控制力略有提升。而即便与目前纯音乐声卡运放驱动的耳机输出相比，HA20的驱动力和声音综合表现都明显落后于它们。具体来说，它的声音表现更是对不起身价，尖锐、干瘪，听感枯瘦乏味，明显不如音源直推来得好听。HA20缺点明显，配合高阻耳机完全没有意义（配合低档音源可以有更大音量……），配合低阻尤其高灵敏耳机时，底噪会比较明显，无法忽略，控制力也较差。可以说，很难为HA20找到一款适配耳机，可以认为，HA20几乎不具备使用价值，这点让人失望透顶。HA20是一款极为失败的产品，但其具有诱惑性，尤其对入门菜鸟来说更是如此，不过理由似乎也不充分，HA20也只不过“远看上去很美”。鉴于以上理由，Soomal给予铁三角AT-HA20耳机放大器以不推荐评级。
