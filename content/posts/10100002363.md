9月13日微软召开Build 2011开发者大会，微软Windows 7部门高级副总裁史蒂文・辛诺夫斯基[Steven Sinofsky]在会上宣布，微软发布Windows 8开发者预览版，官方提供下载。辛诺夫斯基表示，Windows 8的下一个里程碑将是Beta用户测试版，然后是RC(Release Candidate)候选版，然后是RTM版(Release.to.Manufacturing)，最后是GA正式版。他强调，每一步都会有升级，但他并未给出具体时间表。
![Microsoft 微软 BUILD 2011 开发者大会 首日Keynote 展示Windows 8 - 现场](https://images.soomal.cc/images/doc/20110914/00013445.webp)




![Microsoft 微软 BUILD 2011 开发者大会-Windows部门主管Steven Sinofsky](https://images.soomal.cc/images/doc/20110914/00013446.webp)




微软高管今天轮番登场演示了Windows 8的新特性。根据演示，我们可以知道，Windows 8将采用Metro UI，兼容Windows 7应用程序，启动更快、硬件配置要求更低、支持智能手机和平板电脑、支持触控、键盘和鼠标三种输入方式。同时，Windows 8支持ARM和x86架构，内置Windows应用商店、IE 10浏览器，并结合云服务和社交网络等。
![Microsoft 微软 BUILD 2011 开发者大会-展示Windows 8 PC](https://images.soomal.cc/images/doc/20110914/00013447.webp)




![Microsoft 微软 BUILD 2011 开发者大会-展示Windows 8 平板电脑](https://images.soomal.cc/images/doc/20110914/00013448.webp)




![Microsoft 微软 BUILD 2011 开发者大会-展示Windows 8 PC](https://images.soomal.cc/images/doc/20110914/00013449.webp)




截至目前，微软并未正式宣布下一代操作系统的名称及发布时间，微软在公司网站发布的新闻稿件中称“Windows 8”是下一代操作系统的开发代号。值得一提的是，戴尔消费者和中小企业业务总裁史蒂夫・菲利斯(Steve Felice)本周二表示，微软可能在2012年10月推出Windows 8。

Windows 8的新特性
![Microsoft 微软 Windows 8 开发者预览版 - 锁屏页](https://images.soomal.cc/images/doc/20110914/00013438.webp)




“触摸优先”的用户界面：Metro风格：Windows 8推出了一个专为触摸而设计的最新Metro风格界面，能向用户显示重要信息，这个界面同时支持鼠标和键盘。


![Microsoft 微软 Windows 8 开发者预览版 - 开始Start菜单](https://images.soomal.cc/images/doc/20110914/00013441.webp)




![Microsoft 微软 Windows 8 开发者预览版 - 拇指键盘布局](https://images.soomal.cc/images/doc/20110914/00013443.webp)




![Microsoft 微软 Windows 8 开发者预览版 - 触摸键盘布局](https://images.soomal.cc/images/doc/20110914/00013444.webp)




应用沟通：以应用为支持。为Windows 8系统设计的Metro风格应用是用户体验的焦点，能填满整个显示屏，从而让用户不会分心。用户体验可在不同设备之间同步发生：来自于用户最常使用的云服务的所有内容都是活跃的，无论是照片、电邮、日历还是联系人，这使得这些内容能在用户设备中保持最新状态。通过SkyDrive或Windows 8系统的Metro风格应用，用户可在任何地方用任何浏览器来获得其文档、照片和文件。

![Microsoft 微软 Windows 8 开发者预览版 - IE10浏览器](https://images.soomal.cc/images/doc/20110914/00013440.webp)




基本面增强：Windows 8建立在Windows 7系统的牢固基础上，性能、安全性、隐私性和系统可靠性均有所改善。Windows 8减少了内存占用，甚至是在最低端的硬件上也能为应用留出更多空间。

![Microsoft 微软 Windows 8 开发者预览版 - PhotoPicker](https://images.soomal.cc/images/doc/20110914/00013442.webp)




新的开发者机会：Windows Store(Windows商店)：Windows Store将允许开发者在全球任何销售Windows系统的地区出售其应用，无论他们开发的是新游戏还是类似的生产力工具。Windows 8允许用户利用已有技巧和代码，使用最喜欢的程序语言来创造良好的用户体验。丰富的硬件集成带来更加丰富的体验，对游戏来说更是如此。Windows 8拥有DirectX 11接口，允许开发者轻松开发流畅的、无闪烁的全屏游戏。

新一代硬件：一个Windows，多种形状和尺寸。支持ARM芯片组、x86(以及x32和x64)设备、触摸和传感器，这意味着Windows 8能在多种设备之间完美运行，无论是10英寸的平板电脑、笔记本还是27英寸的高清显示器。

永久在线：在Windows 8系统下，新的超薄PC和平板电脑可瞬间启动，只充一次电即可运行一整天，而且永远处于联网状态。下一代系统单晶片支持还将大大提高待机时间和低能耗状态。
