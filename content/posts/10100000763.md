乐之邦在2009年连续发布了数款USB声卡，获得了一定成功。而在PCI声卡领域，并没有重量级的产品推出。



相关文档
《乐之邦 Musiland Moli 茉莉双声道声卡 图集》[作者:Soomal ]

![乐之邦 Musiland Moli 茉莉双声道声卡](https://images.soomal.cc/images/doc/20090902/00002698.webp)



直到9月初，才发布了一款命名为“茉莉”的双声道声卡。乐之邦在内置声卡上，设置有莫邪和轩辕两个系列，而茉莉既不隶属莫邪也不隶属轩辕，它是一个新的系列。我们曾经在其他文章中提过，以名剑名命名声卡，太过于沉重，并不适合，似乎乐之邦听取了这个意见。这款声卡采用白色PCB电路板，并不只是为了好看。Monitor系列温暖的音色获得用户好评，让乐之邦看到了声卡新的发展方向，以音色作为卖点可能会获得更好的市场回报。因此用颜色来代表一个音色成为了新的视觉应用。未来的Monitor系列新品将都采用红色的PCB，最近发布的01 Mini正是如此，而茉莉的音色追求一种清新雅致的格调，所以采用了白色作为音色代言。我们认为这种视觉应用很合适，也更方便用户选择。



![乐之邦 Musiland Moli 茉莉双声道声卡](https://images.soomal.cc/images/doc/20090902/00002699.webp)



茉莉采用刀卡板型设计，以方便安装到薄机箱当中。在标准配件中，并没有提供矮挡板，用户需要另行购买。我们在此建议乐之邦将矮挡板作为标准附件以方便用户。



![乐之邦 Musiland Moli 茉莉双声道声卡-前面板](https://images.soomal.cc/images/doc/20090902/00002700.webp)



茉莉在I/O设计上舍弃了一切硬件输入接口，保留了耳机、左右声道RCA独立输出以及光纤输出。这与节奏坦克的小夜曲声卡非常相似，小夜曲的成功，也证明了这种纯输出声卡是有市场的。从《乐之邦 Musiland Moli 茉莉双声道声卡 图集》[作者:Soomal ]
的点击量与留言来看，这种纯输出声卡的用户关注度非常高。我们一贯主张音频设备的设计应该从简，可以不要的一律不要。



![乐之邦 Musiland Moli 茉莉双声道声卡-VIA Tremor(VT1723) I/O控制器](https://images.soomal.cc/images/doc/20090902/00002701.webp)



茉莉采用的VIA Tremor(VT1723) I/O控制器，这款控制器是Envy24系列中较为低端的型号，但并非低端低能，这款芯片具有与Envy24高端芯片相同的数字I/O质量，支持24bit/192kHz的输入输出，因此用Tremor设计一款双声道输出的声卡并不会有性能上的损失，除非设计失误。



![乐之邦 Musiland Moli 茉莉双声道声卡-PCM1793](https://images.soomal.cc/images/doc/20090902/00002703.webp)



市面上有数款基于Tremor的声卡，大部分的模拟输出依赖一颗VIA的Codec完成，而茉莉并没有走这样的设计路线，它采用了一颗BB出品的PCM1793 DAC芯片。这款DAC芯片支持24bit/192kHz，信噪比为113dB，这是一颗广受好评的中档DAC芯片，用于声卡设计，我们认为它不会构成音质瓶颈。



![乐之邦 Musiland Moli 茉莉双声道声卡-JRC4580运算放大器](https://images.soomal.cc/images/doc/20090902/00002705.webp)



模拟输出的运算放大器采用JRC4580，这颗运放芯片能在很多中高档声卡上找到，它的特点就是解析力出色，音色中性平淡。



![乐之邦 Musiland Moli 茉莉双声道声卡-集成的耳机放大器](https://images.soomal.cc/images/doc/20090902/00002704.webp)



茉莉卡上还集成了一个由D669与B649对管构成的耳机放大器，这种设计在很多中高档声卡有所运用。虽然元器件相似，但这些耳机放大器的音质也有高有低，整体而言，这种设计适合驱动中低阻抗的耳机。



![乐之邦 Musiland Moli 茉莉双声道声卡-富士通高见泽继电器](https://images.soomal.cc/images/doc/20090902/00002706.webp)



在茉莉的驱动当中，可以通过控制界面来切换耳机或者线性输出，切换时会听到嗒的一声，这是这颗继电器在工作。小夜曲也有集成了耳机放大器，但它却不能通过程序切换，耳机与线性输出是同步，这点上，茉莉显然要更方便一些。



![乐之邦 Musiland Moli 茉莉双声道声卡-局部](https://images.soomal.cc/images/doc/20090902/00002707.webp)



在茉莉声卡的右上角，有这么两个插座，一个10Pin一个2Pin，2Pin的为S/PDIF输出，它用于连接显卡，然后通过HDMI输出。不要简单的认为外接一个RCA插座就可以实现同轴输出，因为这组S/PDIF的信号电平并不是标准电平，会有部分解码器识别失败。而10Pin的插座是子卡插座，乐之邦计划在未来的PCI声卡设计当中，采用统一接口的子卡，用于各个系列的声卡的组合搭配。



![乐之邦 Musiland Moli 茉莉双声道声卡-FPGA](https://images.soomal.cc/images/doc/20090902/00002702.webp)



基于Envy24系列芯片的声卡，大部分都有两颗晶振器，分别用于产生44.1kHz和48kHz以及与它们相关的频率。而茉莉却只用到了一颗晶振，其秘密就在这颗FPGA。茉莉利用这颗Xilinx Spartan XC3S50A来合成44.1kHz以及相关频率，利用FPGA内部的时钟管理器来提高精度。并对音频信号时序进行了重整，降低信号的抖动，根据乐之邦的说法，这颗芯片还提供了S/PDIF的编码运算任务，能支持176.4kHz，而这个频率是Envy24系列芯片内置S/PDIF发射器所不支持的（I2S方式能支持）。这部分显然是借鉴了Monitor系列设计中的成功经验。通过这个设计，可以看出，Tremor I/O控制器更像是一颗PCI的界面芯片，或者说乐之邦对Envy24系列芯片的性能已经开始有所不满，需要用FPGA来打打补丁。这不禁给人一点遐想，乐之邦什么时候用FPGA完全替代I/O控制器呢？就像RME Hammerfall DSP 9632那样。



由于使用了不一样的硬件设计，因此驱动上会有所不同。我们使用的操作系统是Windows XP SP3，驱动很顺利的安装完毕。



![茉莉驱动界面](https://images.soomal.cc/images/doc/20090912/00002745.webp)



这个版本的驱动的控制界面与Monitor系列声卡的几乎一样，简单明了。虽然与莫邪、轩辕系列一样使用了Envy24系列的I/O控制器，但茉莉的驱动并没有使用乐之邦引以为荣的沙林驱动，因为沙林太过复杂，而茉莉却是乐之邦成立以来，功能最简单的一张PCI声卡。这个驱动操作起来很方便，就是美工设计太简陋了点。



![Foobar2000 - 使用茉莉声卡ASIO输出](https://images.soomal.cc/images/doc/20090912/00002750.webp)



茉莉支持ASIO，使用Foobar2000外加第三方ASIO插件测试通过。每次提到ASIO，我们都会强调，对以播放而言，ASIO的意义不大，对于这种纯输出声卡而言，ASIO是没有无所谓，有了也无所谓。



来看看RMAA测试的结果，由于RMAA并不是精确测试软件，其测试成绩仅供参考，请勿盲目对比。



![RME Hammerfall DSP 9632 声卡](https://images.soomal.cc/images/doc/20090502/00001672.webp)



录入端采用的是 RME Hammerfall DSP 9632 声卡。



![44.1kHz、16位条件下的茉莉线性输出、HDSP9632线性输入](https://images.soomal.cc/images/doc/20090912/00002746.webp)



| 测试项目 | 结果 |
| --- | --- |
| 噪声水平, dB (A): | -95.2 |
| 动态范围, dB (A): | 95.1 |
| 总谐波失真, %: | 0.0071 |
| 互调失真, %: | 0.011 |
| 立体声分离度, dB: | -95.7 |



测试得到一条平直的响应曲线，在高频段，有0.2dB左右的衰减，这对高频无可察觉的影响。



![44.1kHz、16位条件下的茉莉线性输出、HDSP9632线性输入](https://images.soomal.cc/images/doc/20090912/00002747.webp)



查看总谐波失真，会发现一些貌似成规律排列的谐波，由于这些谐波峰值很低，它不会对测试结果产生影响，但它们会悄悄的改变音色。



![44.1kHz、16位条件下的茉莉光纤输出、HDSP9632光纤输入](https://images.soomal.cc/images/doc/20090912/00002748.webp)



| 测试项目 | 结果 |
| --- | --- |
| 噪声水平, dB (A): | -95.8 |
| 动态范围, dB (A): | 95.7 |
| 总谐波失真, %: | 0.0006 |
| 互调失真, %: | 0.0049 |
| 立体声分离度, dB: | -96.3 |

不出意料，得到一条绝对平直的曲线。



![44.1kHz、16位条件下的茉莉光纤输出、HDSP9632光纤输入](https://images.soomal.cc/images/doc/20090912/00002749.webp)



这是S/PDIF 输出时的总谐波失真状况，你可以对比一下上面那张图。48kHz/16bit条件测试下的结果几乎一致，不再另行粘贴。乐之邦号称茉莉支持176.4kHz的S/PDIF信号输出，经过测试，确实可以通过。但我们不得不说的是，我们以前并没有留意过Envy24 系列I/O控制器内置的S/PDIF发射器是否支持176.4kHz，请有条件的朋友测试一下。另外，176.4kHz并不是一个很重要的采样率，很少有这样音乐节目，所以是否支持176.4kHz的S/PDIF输出，并不是购买声卡时的参考项目，你需要关心的依然是常用采样率下的音质表现。



在实听时，我们尽可能的和更多的耳机或者音箱，这里不再一一列举，这些回放设备的价格从200-2000之间。



茉莉的高音是一种轻盈的高音，具有良好的瞬态表现，以及出色的解析力，它的解析力并没有带来毛毛糙糙的感觉，听感上是比较细致内敛的那种。很适合表现弹拨类乐器，表现拉弦类乐器，例如小提琴，也有上佳的表现。中频婉约细腻，适合表达那种甜蜜蜜的女声，例如邓丽君、李烁这类的。它有着良好的解析力和结像力，欣赏人声曲子时，可以更精确的表现歌者的口型。低频部分，具有较好弹性以及层次感，表现各类打击乐都有不错的表现。对于小编制的音乐，茉莉都能拿捏得有滋有味，对于那种宣泄的重金属，茉莉不合适，不够张扬。有得有失，我们鼓励厂商设计有风格的声音，中性化的声音越来越让人审美疲劳。从音色角度说，茉莉非常像我们以前测评过的轩辕II（《乐之邦 Musiland 轩辕II X-SwordII Pro-HD 声卡测评报告 》[作者:夏昆冈 ]
），但还欠缺一些大度与灵气。差异主要出在低频，单独去评论低频，茉莉很优秀，但茉莉整体追求的是清新雅致的一种风格。从声音美学上来说，茉莉的低频力度还可以弱一点，这样整体的平衡感会更好，也会因此显得更加大气。



茉莉集成了耳机放大器，虽然采用了常见的D669与B649对管设计，但它的表现却与其他设计不同，茉莉的耳机驱动力偏弱，但声底非常干净，使用高灵敏度的入耳式耳塞也听不到噪声，耳机输出与线性输出的风格基本一致，由于有继电器切换，使用起来很方便。我们测试了几条耳机，阻抗100欧以下的耳机能良好支持，通常D669与B649的耳机放大器，能较好驱动150欧左右的耳机，这算有得有失吧。



虽然茉莉还算不上很完美，但对于一款398元的产品，已经做得很不错了。茉莉给出了绝对拿得出手的硬件配置，还达到了较高的音质表现，这是竞争带来的实惠，Soomal给予此产品强烈推荐评级。
