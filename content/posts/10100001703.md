清逸伦是北京的一家音响器材公司，但说起清华吴刚，可能知名度更高一些，实际上，这家企业和清华并没有明确的人事、从属或者被投资关系。对于发烧友来说，G&W T-2.6F同样是一个非常知名的型号，在国产品牌耳放中它2000多元的价格也属于中高端配置，无数森海塞尔HD600和HD650的烧友甚至歌德耳机的烧友都会被推荐选用这款耳机放大器。

![清逸伦 G&W T-2.6F耳机放大器-与全尺寸头戴式耳机大小对比](https://images.soomal.cc/images/doc/20101006/00007489.webp)




硬件设计
清逸伦G&W T-2.6F[以下简称T-2.6F]是一台耳机放大器，体积非常大，如果你希望摆放到桌面使用，需要占用较大的空间，而且这种造型并不适合叠置。大体积不能算优点，但也算不上缺点，如果你对它的体积没有概念，不妨看看它与全尺寸耳机的大小对比。

![清逸伦 G&W T-2.6F耳机放大器](https://images.soomal.cc/images/doc/20101006/00007486.webp)




T-2.6F采用的不锈钢机壳，前后面板采用拉丝铝板，其中前面板厚度为10毫米，加工较为精致，看上去颇有档次。

![清逸伦 G&W T-2.6F耳机放大器](https://images.soomal.cc/images/doc/20101006/00007487.webp)




前面板上放置了电源开关、音量旋钮以及一个6.25mm的耳机输出插座。

![清逸伦 G&W T-2.6F耳机放大器-后面板](https://images.soomal.cc/images/doc/20101006/00007488.webp)




后面板上有两组RCA，一组输入一组输出，T-2.6F可以作为前级放大器使用。

![清逸伦 G&W T-2.6F耳机放大器-电路板](https://images.soomal.cc/images/doc/20101006/00007491.webp)




拆掉机箱底部盖板，就能看到功放板。内部不乏猛料，例如发烧友推崇的大红WIMA电容、ALPS蓝壳电位器等。更多细节照片可以点击下方图集。

相关图集
《清逸伦 G&W T-2.6F耳机放大器拆解 图集》[作者:Soomal ]

T-2.6F的功放板结构很清晰，后半部分为电源处理电路。前半部分是功放电路，又分为前级和后级部分。但这个结构却谈不上最优化，你能看到很多内部连线，例如从连接到前面板6.25mm耳机输出和飞越电源处理电路连接到后面板RCA前级输出的信号线等等，这些连线影响美观之外，也容易引入噪声。

![清逸伦 G&W T-2.6F耳机放大器-捷克Tesla E88CC 真空管](https://images.soomal.cc/images/doc/20101006/00007490.webp)




T-2.6F前级采用真空管，它在此款机器中相当于运算放大器的作用。T-2.6F的默认配置是6922EH，也可以是ECC88、6H23。我们手头的这款配置的是捷克Tesla E88CC。相比晶体管和集成电路，真空管往往具有更加温暖的音色，真空管还具有压限效果，所谓压限即压缩和限制的合成，我们曾经撰写过相关文章，请参阅《听音入门 声音的压限处理》[作者:夏昆冈 ]
一文。

在使用真空管之前，需要了解真空管的优劣，如果设计体现不出真空管的优势，那么这个设计可以视为失败的。
![清逸伦 G&W T-2.6F耳机放大器-J313 K2013功率管](https://images.soomal.cc/images/doc/20101006/00007495.webp)




后级部分采用J313/K2013中功率对管，这点与官方资料并不相同，官方资料显示使用的是K214/J77。J313/K2013安装比较特别，对应位置的PCB留下镂空位置，功率管放倒后直接安装到机顶散热器底部，这种散热方式的效率应该能有保证。

T-2.6F前级为真空管，后级为晶体管，这种设计被称为“前胆后石”，也被戏称为“胆结石”，它可能同时具有两者的优点与缺点，优秀的设计应该是扬长避短，让两者的优点更加突出。

客观测试与分析
![EMU Emulator X Studio 桌面采样系统](https://images.soomal.cc/images/doc/20100604/00005777.webp)




关于耳机放大器的客观测试，我们还是选用专业级声卡录入的形式，结合常见的分析方式，提供直观的测量结果。警告：民用声卡用户请勿尝试这种方式测试，我们无法确保信号过载时不会对声卡产生伤害。

这种方式测试有其局限性，它与实际使用有一定差异，实际使用时会有不同的负载。但它反应的数据依然有参考性，尤其对左右声道的平衡性、分离度等指标的分析，能更加直观。

首先我们来看一下RMAA测试数据。我们使用MD11作为前端音源输入至耳机放大器，使用E-MU Emulator X Studio进行录音。


| 测试项目 | 结果 |
| --- | --- |
| 噪声水平, dB (A): | -94.9 |
| 动态范围, dB (A): | 95.0 |
| 总谐波失真, %: | 0.264 |
| 互调失真, %: | 0.242 |
| 立体声分离度, dB: | -87.8 |



![清逸伦 G&W T-2.6F RMAA频响曲线图](https://images.soomal.cc/images/doc/20101112/00008242.webp)




![清逸伦 G&W T-2.6F RMAA立体声分离度图](https://images.soomal.cc/images/doc/20101112/00008243.webp)




从立体声分离度测试数据来看，RMAA的数据统计会给出一个数字值和一条曲线，如我们在这里得到的立体声分离度是-87.8dB，这个值只代表了在1KHz时立体声分离度，而从上图立体声分离度频谱曲线来看它的差别会随频率变化而发生变化。所以我们将进一步进行以下测试。


关于立体声分离度测试以及音频频谱分析，我们在这里专门说一下我们的测试方法，以便大家来读懂文字和图表。

测试的方法非常简单，我们用频率发生器产生一个20Hz-20KHz的正弦波扫频信号，我们使用CoolEdit软件和负责录音的声卡对待测试设备的输出进行录音。如果在RMAA中，我们看到出现立体声分离度出现问题，则进一步进行以下的测试。


将耳机放大器（待测设备）的输入切断一个声道，此时用音源播放频率发生器发生的标准信号，并用软件开始从耳机放大器的耳机输出口进行录音，此时录音时双声道输入录音用声卡的。我们会得到以下一个波形文件。

通过CoolEdit2000，我们可以以不同的方式来分析这个波形。首先，是大家常见的波形图（绿色波形）。这张图片的横轴是时间轴，纵轴是信号强度（振幅），波形直接反应信号的振幅。大家可以看到（点击放大图片浏览更清晰）由于我们停止了一个声道的信号输出，该声道本应该没有信号最多也只是存在一些底噪，但图中看到，右中心轴它随着频率升高也有一定的信号。

![清逸伦 G&W T-2.6F 立体声分离度测试（右声道关闭）](https://images.soomal.cc/images/doc/20101112/00008237.webp)




我们再用光谱的方式来分析同样的信号。光谱图中，坐标轴横轴仍为时间轴，纵轴是信号的频率，而信号的强度靠光谱的能量来表现。黑色表示完全静音，亮白色表示强度大。颜色越亮，信号强度越大，一般我们会看到蓝红色的底噪、谐波失真，而如果发生串音也一目了然。

![立体声分离度测试 20Hz-20KHz 正弦波扫频光谱分析图说明](https://images.soomal.cc/images/doc/20101112/00008244.webp)




在整个坐标图范围内，底色很暗，可以看到一条随时间轴具有一定斜率的亮白色直线，它就是我们播放的主要信号。但我们可以从图中看到，与这条直线不同斜率的更暗更细（呈蓝色、暗红色）的直线，这就是谐波失真。而我们看另一个声道，这里也有一条红色的同样斜率的信号，它的亮度会有所不同代表信号不同，这里我们已经关闭了该声道输入，这就是另一个声道发生了串音。而它的强度还会随频率变化。

![清逸伦 G&W T-2.6F 立体声分离度测试频谱对比分析图（左声道关闭）](https://images.soomal.cc/images/doc/20101112/00008236.webp)




最后一张是左右两个声道能量对比频谱。横坐标是频率而纵坐标是频率对应的能量（分贝数），两个颜色区域对比可以看到串音的强度是随频率的变化而变化的。

下面我们来具体分析一下清逸伦T-2.6F的输出变化，我们可以看到无论是关闭左声道或右声道输入，都会发生明显的串音。而且从光谱分析图来看，图中上方的一个声道（左声道）有明显的低频噪声而右声道则基本没有。观察左右声道同时录音的光谱图，可以发现两个声道的谐波差别较大，谐波的强度差别也很大（左声道）。这样的结果与RMAA测试结果吻合，只不过RMAA的数据取样不够合理，图表相对不够直观。

主观听音测评
我们使用乐之邦 MD11解码器（Monitor 01 PS Mulink输入）作为主要测评音源，长征2解码器与MINI DAC解码器作为参考。AKG K701（阻抗62欧姆，煲机350小时以上）和飞利浦SHP9000（阻抗32欧姆，煲机150小时已OK）作为低阻版测试耳机，森海塞尔HD650（阻抗300欧姆，煲机300小时左右）和拜亚动力DT880（600欧姆版，煲机150小时以上）作为高阻抗版测试耳机。



飞利浦SHP9000
飞利浦SHP9000煲机150小时，声音已经煲开，它的声音风格松软均衡，但中频部分相对不容易驱动，阻抗很低只有32欧姆，和AKG K701某些方面有些类似，但也有明显差别（具体对比见以后测评文章）。在清逸伦T2.6F下，SHP9000的高频较为柔软，但声场明显不好，解析力一般，弦乐如小提琴表现偏柔和，大动态时细节不佳明显无力，瞬态细节不够好。中频有些塌陷，人声明显偏薄。低频控制力一般，力度差，但相对与普通声卡层次和解析力稍有提升，大动态不至于混乱。与MD11耳机输出相比，结像和声场清逸伦T2.6F表现不佳。小动态音乐表现有一定的味道，保持了SHP9000一定的特色，但没有发挥出SHP9000高频细节的优势，中频松弛，解析力也受到了影响。对于特大动态表现较差，基本属于毫无用处的状态。

AKG K701
AKG K701高频拥有很好的解析力，且只有62欧姆的阻抗，虽然低阻抗大尺寸耳机同样难以伺候，但它与高阻抗耳机“没有推好”的状态是不一样的。在清逸伦T2.6F上，K701的表现状态更类似飞利浦SHP9000，高频解析力较好，但中高频部分开始控制不佳，小提琴和年轻女声声音有些失真，小提琴的声音虽然动态较好，但层次较差，声音有被挤压的感觉；而年轻女声则有严重的鼻音俗称“感冒声”。而在成熟女声方面，K701下清逸伦T2.6F表现更好，声音稍饱满一些。但在交响乐等大动态音乐表现上，仍显得非常无力。与SHP9000下声音的塌陷以及HD650上声音粗糙偏薄不同，K701下在大动态交响乐时中频量感都好于前两者，动态也较好，但层次变差，声音较乱。至于低频表现，K701最为难以驱动，解析力、动态和量感都较差。

森海塞尔HD650
森海塞尔HD650是这几款耳机中较难驱动的一个，声音解析力很好但又是高阻抗的配置。在这套组合下，HD650的声音明显粗糙发干，中频和高频显得更加无力，HD650的高频解析力相当出色音色也相当独特，在T2.6F上仍残留一些影子，但即便与MD11耳机直接输出相比，高频的表现已经下降了几个档次。T2.6F在高阻耳机上的不足体现的更为明显，中频的解析力尚可，但声音更薄，动态严重不足。在交响乐等大动态表现上，已经明显不够好。低频方面，HD650的量感仍在，但声音更加偏紧，解析力均一般，很难表现出大动态的应有气势。


拜亚动力DT800 600欧姆版
拜亚动力DT880这款耳机目前煲机时间较短只有150余小时，但它在清逸伦T2.6F上表现再次映正了HD650表现不佳的问题和趋势，同时以上各种问题变得更加明显，尤其是高频开始出现的问题。清逸伦T2.6F驱动DT880更为吃力，声音粗糙感没有HD650那么明显，但高频量感受到很明显影响，高频已经开始明显偏暗，整个声音动态变得很小，DT880此时听上去就像一个很小很普通的耳机。此时声音表现基本惨不忍睹……



对于高阻抗耳机，清逸伦T2.6F拥有如此强大的用料配置和供电支撑，却非常不适合高阻抗耳机。相对而言，高阻抗耳机在T2.6F上的表现相当差，只能在小音量下欣赏人声、弦乐独奏等小动态音乐，而此时它的高频细节和声场还不如一块普通的Monitor 02 US级别的声卡（在DT880上尤为明显），T2.6F在高阻抗耳机上的表现根本不符合“耳机放大器”的称号，只是在声音风格上做了处理，但效果却不佳。

在低阻抗耳机上，飞利浦SHP9000整体表现要好于K701，而K701下的解析力要好于SHP9000，但大动态时K701表现出层次不佳，声音失控有些混乱；而SHP9000在大动态音乐中只是显得动态不足，声音偏软。对于低阻抗两款耳机，清逸伦TT2.6F的中频和高频控制力仍一般，在高解析力下的K701下声音粗糙而在SHP9000下声音偏软。低频方面，SHP9000下的表现量感和动态都更好一些，K701的表现则不值一提。

总体而言，清逸伦T2.6F声音较为粗糙，存在左右声道可闻的差异（也许是个体因素），声场表现不好明显偏窄，对于我们使用几个耳机来看，作为耳机放大器它却难以胜任“放大”的本职工作，更谈不上什么胆味。对于高阻抗耳机表现出动态差、声音暗等问题；对于低阻抗耳机又出现典型的声音粗糙，层次差，声音乱的问题。

这个结果实在让我们感到十分意外，一个用料豪华，如此巨大身材的耳放表现的却是弱不禁风，毫无生气的声音，仅仅在低阻抗耳机下可以听听动态很小的小品类音乐。鉴于清逸伦T2.6F整体较差声音表现，不合理的定价和相对认真的做工，我们给予清逸伦T2.6F以不推荐评级。
作为国内耳放中知名型号，中国“胆耳放”的代表作品，清逸伦T2.6F并不应该代表国内耳机放大器的水平，以及相应的定位以及售价，也不值得业内同行去对比参考。它的表现甚至给国外品牌产品卖高价做了铺垫。如果以它的表现和定价为标准，那么SOLO的表现和售价是不是就理所应当？随着更多耳放加入，我们会增加更多对比，对于清逸伦T2.6F的评价并没有结束。我们将于11月18日发布SOLO耳放的测评报告。
