![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]](https://images.soomal.cc/images/doc/20120812/00021901.webp)



T1是IRIVER艾利和在2011年发布的一款低端便携式MP3播放器，目前售价在145元。与常见的便携播放器不同，艾利和T1没有内置存储空间，必须通过安装Micro SD卡[TF卡]来提供存储空间[最大支持32GB，附件配有4GB存储卡]。艾利和官方将T1命名为“Combo Stick”，意为它是“MP3 Player”“Micro SD Reader”“USB Memory Stick”三者优点相融合的一款产品，即：T1拥有闪盘的外形，是一款MP3便携播放器，也是一款Micro SD读卡器。



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]-对比飞利浦GoGear](https://images.soomal.cc/images/doc/20120812/00021911.webp)



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]](https://images.soomal.cc/images/doc/20120812/00021902.webp)



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]](https://images.soomal.cc/images/doc/20120812/00021906.webp)



我们先看机身设计，艾利和T1采用类似于普通闪盘的造型，它的体积较小与飞利浦SoundClip相仿。同时，其重量也比较轻[含存储卡重12g]。T1采用塑料外壳，外壳的上表面设计简洁，印有IRIVER LOGO字样。在中央部分靠下的部分，它还内置有一个双色LED指示灯。而在机身上侧面，有开关按键和一个一体化长条按键。



使用时，T1具备耳机插入识别功能，即插入耳机后长按开关键才能够开机，开机后蓝色指示灯常亮。而在开机状态下，长按开关键或者拔出耳机均可以关机，短按开机键可以播放和暂停当前乐曲。在正常播放时，蓝色指示灯会不断的闪烁。长条按键分为左、右两个区域，长按其中任一区域可以调整音量的大小，蓝色指示灯依旧闪烁。而短按则可以进行上一首/下一首曲目切换，此时指示灯快闪一次，然后恢复原来的闪烁状态。在充电状态时，红色指示灯亮起、闪烁，充满后红灯常亮提示。



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]-插入MicroSD存储卡](https://images.soomal.cc/images/doc/20120812/00021907.webp)



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]-RDC7006主控芯片](https://images.soomal.cc/images/doc/20120812/00021909.webp)



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]-NANSI 隆智 N25S80-75HE 存储卡USB管理芯片](https://images.soomal.cc/images/doc/20120812/00021910.webp)



艾利和T1采用直插式USB设计，USB接口位于机身左侧，而接口上方、机身的内侧设计有隐藏式Micro SD[TF]卡槽。我们对其进行拆解发现，T1采用RDC7006主控芯片和NANSI 隆智 N25S80-75HE 存储、USB管理芯片。其中，RDC7006常用于一些低价的插卡式MP3或者插卡耳机中。而这些产品通常没有内置闪存，需要安装外部存储设备来提供数据存储空间。如此，它们可以当作读卡器来使用。



![ATTO Disk Benchmark测试成绩-IRIVER 艾利和 T1 便携数字播放器](https://images.soomal.cc/images/doc/20120825/00022228.webp)



![ATTO Disk Benchmark测试成绩-某品牌读卡器](https://images.soomal.cc/images/doc/20120825/00022229.webp)



据官方资料称：“T1内置最新控制芯片，采用USB 2.0接口，具备高速存储特性。”我们使用附件中的Micro SD卡、ATTO Disk Benchmark软件进行读写测试，T1的平均写入速度接近4.5MB/秒，平均读取速度接近7.5MB/秒。而我们自己使用的读卡器的平均写入速度在6MB/秒，平均读取速度在9MB/秒。T1的表现如何？一目了然。



在续航能力上，T1内置90mAh锂电池，官方标称连续播放时间为3小时[实测接近]，续航较短。而相对的，它的充电时间较长，大约一小时才可以完全充满。

从客观测试数据来看，艾利和T1的表现很差。仔细分析RMAA的测试图表并结合我们的频率扫描频谱图片分析来看。它的信噪比只有-81dB，频响曲线一直存在明显抖动，虽然MP3测试文件会对测试造成一定影响，但也比我们曾经使用MP3测试文件进行测试的结果差得多[例如索尼B152F、B162F等]。而无论是RMAA的具体噪声分布图，还是频谱扫描图，噪声都过大。而互调失真测试分布图已经差的一塌糊涂，惨不忍睹。其实其他客观测试图表的表现也是我们见过最差，最离谱的。



![IRIVER 艾利和 T1 便携数字播放器[兼读卡器功能]-耳机](https://images.soomal.cc/images/doc/20120812/00021912.webp)



主观听感



艾利和T1原配一副耳塞式耳机，这款耳机易于驱动，但品质较差。在原配耳机下，IRIVERT1的低频没有量感，下潜和力度也都不好；高频部分声音干涩，解析力差；而中频的表现相对好一些，声音相对厚实，但解析力较差，声音缺乏细节。我们又使用了Aurvana Air、漫步者H260、声美E10等耳机进行试听，T1的结像力不好，整体的解析力较差，声音比较粗糙、没有层次。同时，T1的动态表现和控制力也都很一般，在稍大动态时，它的声音会出现失真。



总结



艾利和T1的声音表现太差，是我们测试过的便携播放器中表现最差的一款。而且相比大多数的手机，其音质也存在明显的差距。作为MP3的知名品牌，艾利和出品这样一款播放器令人失望。与百元级便携播放器产品相比，大概只有花50元装上一块16GB存储卡时才能找到一点心理安慰？当然，在特殊应用中也许它的读卡器应用让你可以分享一张存储卡中的数据，但这种机会和应用似乎越来越少，而一款TF专门读卡器售价一般不到10元。



T1号称是“Combo Stick”，但主要功能显然还是一款播放器，遗憾的是本职工作方面做得实在太差。在145元左右价位，我们有很多入门播放器可选，而IRIVER T1已经不是“性价比”不够好的问题。读卡器的功能是否需要？用户很容易判断。而IRIVER的播放器的音质如此离谱，显然出乎我们和很多消费者的意料。Soomal给予艾利和T1便携播放器强烈不推荐评级。
