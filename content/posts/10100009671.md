自2021年起对测评规则的一些调整。

1、不再手工干预任何影响色彩准确度的参数，仅仅选择预设色彩模式。
2、在色彩准度测试中增加了100%红蓝绿青黄洋红的测试色卡。
3、P3色域的百分比值不再对应sRGB色域，而是对应P3色域，所以百分比值和以往机型没有对比性。

修改规则的原因是要增加难度，鼓励厂商提供直接能用好用的预设组，而不是让用户自己去校正。

![OPPO Find N 折叠智能手机](https://images.soomal.cc/images/doc/20220520/00098469_01.webp)



手机屏幕发展到7寸大小基本就认为快触碰到直板设计的天花板了，但消费者还是希望更大的屏幕，所以出现了各种尝鲜的设计。OPPO推出过卷轴式的概念机，微软推出过商品化的折叠机身但不折屏的内双屏的Surface duo，折叠屏幕的机型也分为内折和外折两个流派，大部分厂商选择的是内外双屏的内折设计，本文主角OPPO Find N就是这样的机型。“从尝鲜，到常用”是OPPO Find N的宣传口号，是OPPO的第一款商品化折叠机型，强调的是实用性。 在此之前，卷轴式概念机，更多是为了炫技。



Find N采用双屏设计，即内外屏设计，折叠状态朝外的即为外屏。外屏5.49英寸，分辨率1972*988，内屏7.1英寸，分辨率1792*1920。内外屏均采用AMOLED面板，但刷新率有差异，其中外屏刷新率60Hz，而内屏支持最高120Hz自适应刷新率，玻璃盖板也有差别，外屏为康宁大猩猩，而内屏则是肖特UTG。在Find N的产品设计中，内屏被定义为主屏。



![OPPO Find N 折叠智能手机-拟椎式铰链[来自官网]](https://images.soomal.cc/images/doc/20220527/00098602.webp)



早期的折叠机型，常常会出现屏幕死折甚至断裂的现象，它对读屏体验过于负面，也影响严重影响寿命。目前的解决之道是从铰链设计入手，让屏幕以水滴型弯曲，避免硬生生的对折。Find N的铰链采用拟椎设计，铰链中间部分模拟脊椎体，两侧类似横突的部位则是铰链的杆部，这种设计可以避免屏幕对折出现死折。也能降低折痕深度。实际使用，会发现这个设计确实不赖，Find N开箱的新机状态几乎是完全平整的，使用一段时间[超过一个月]后，屏幕折痕依然很浅，仔细观察折痕是非常非常浅M形态。



![OPPO Find N 折叠智能手机-屏幕刷新率设置](https://images.soomal.cc/images/doc/20220527/00098608_01.webp)



![OPPO Find N 折叠智能手机-屏幕色彩模式设置](https://images.soomal.cc/images/doc/20220527/00098609_01.webp)



Find N操作系统采用ColorOS，屏幕相关设置与其他基于ColorOS的机型大同小异。Find N在使用主屏，也就是内屏的状态下，可以设置刷新率，使用外屏时，只能60Hz。色彩模式具有生动、自然、专业三个模式，其中专业下又有影院和鲜艳两个子模式。生动模式色温高，很多消费者喜欢高色温的观感。自然和专业模式都基于D65白点，其中自然模式是基于sRGB色域，专业[影院]基于P3色域，而专业[鲜艳]就是面板放飞模式了。Find N似乎不具备根据图片色彩描述信息切换色域的能力，在下面的测试中，会根据测试项目手动切换色彩模式。

根据新规则，已经不再允许干预其他色彩可调参数，这实际上加大了获得高分的难度，我们希望鼓励厂商直接将最优化的预设参数提供给消费者。按照惯例，屏幕的客观测试使用Portrait Displays Calman完成。

使用体验：在一般环境中，Find N 外屏最大亮度为511 cd/m2，强光下触发全局峰值亮度，实测为803 cd/m2，相比顶级机型900-1000+ cd/m2的水平，略逊一筹，但800 cd/m2的水平，也能实现优良的户外可视性，强光下读屏、取景并不吃力。内屏在一般环境下的最大亮度为489 cd/m2，强光下触发真正的峰值，实测787 cd/m2。当内外屏幕亮度约等于100cd/m2时，都会有可感知频闪，其中内屏，也就是主屏，闪烁强度弱一些，读屏舒适度也相对好一点。
阅读、看图、炒股这些类型的应用，对屏幕尺寸有着天然的高度依赖，内屏更大的尺寸能明显的提升这类应用的体验。对于视频类应用，大尺寸也能提升观看舒适度，即便它因为长宽比的原因无法用到太高的屏幕利用率，但还是会觉得更爽。国内不少电商软件并没有主动适配折叠屏手机，三星Galaxy Z Fold3在用内屏使用这类软件时体验比较糟糕，而OPPO的策略是分屏显示两个页面窗口，以提升使用效率。利用手机的半折叠状态，可以实现部分软件的悬停显示，类似于一台小笔记本似的。Find N也支持半折叠状态下低机位取景拍摄，拓展了拍摄场景。

专业色彩：Find N外屏内屏在自然或专业模式下实测色温分别为6718K和6518K，内屏更加接近标准的6500K。平均灰阶分别为2.4和1.7，外屏GAMMA2.1，内屏为GAMMA2.2，在灰阶显示上，内屏能更平滑的呈现平滑的平滑。sRGB覆盖率分别为95.2%和99%，平均色偏分别为1.7和2，最大色偏分别为3.6和3.9。P3覆盖率分别为96.2%和96.9%，平均色偏分别为1.8和1.3，最大色偏分别为4.2和3.1。内屏，也就是主屏，在专业色彩表现方面，要优于外屏，加上两者还存在刷新率方面的差距，所以内外屏切换在观感上会有一点差别。

Find N的屏幕表现中规中矩，内屏能够带来优良的大屏体验，并且对常有APP和部分应用场景进行了针对性优化，所以，内屏你会愿意用，在使用某些优化较好的APP时，会更爱用内屏呈现。而外屏在一般应用下就像普通直板机那样，不会像三星Galaxy Z Fold那样非常规的长宽比带来部分应用体验明显下滑的问题。Find N还有个既是缺点又是优点的特征，就是它的体积较小，限制了屏幕更大化，但也带来了相对轻巧的优势。



注：屏幕面板的一致性不及其他元器件那么高，有些厂商甚至会同时使用多种不同厂商的面板，所以测评数据仅供参考。
