小米在TWS产品线上相当努力，产品更迭速度快得惊人，被定位“年度大作”的降噪耳机Pro[FlipBuds Pro]发布没多久，降噪耳机3 Pro又发布了，降噪耳机3 Pro英文名Buds 3 Pro，其实与FlipBuds Pro并不属于同一系列，但Buds 3 Pro上市之后，FlipBuds Pro似乎悄然退市了。



## 软硬件设计



![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20211222/00096835_01.webp)



![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20211222/00096837_01.webp)



Buds 3 Pro 共有三种色款，分别为黑白绿，电池盒卵形基本造型，比较扁平，表面喷砂工艺处理，观感不错，但触感不算太好。翻盖阻尼适合，合页有较为明显的松动感，通过磁力吸合顶盖，合盖发声清脆。耳机的壳体的材质与电池盒相同，壳体大部分为磨砂处理，与电池盒的一致，耳壳外侧有亮面饰条配色，增加了色彩与光泽变化。



![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机-戴入耳模](https://images.soomal.cc/images/doc/20211222/00096840_01.webp)



![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机-与小米FlipBuds Pro](https://images.soomal.cc/images/doc/20211222/00096841_01.webp)



耳机采用豆芽基本型，造型上并无很特别的设计，有似曾相识的感觉，与FlipBuds Pro相比，显得瘦长一点。配色增加了点识别度。



![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机-扬声器结构[截图自官网]](https://images.soomal.cc/images/doc/20211225/00096864.webp)



“HiFi级音质”是Buds 3 Pro的卖点，它采用的扬声器，使用了DLC振膜，严格说，其实是涂层，DLC 是Diamond-like Carbon的缩写，中文意思是类金刚石碳，也有翻译成类钻石的，主要是显得高大上。DLC具有钻石的高硬度特性，使用DLC涂层的振膜可以提升刚性，而高频响应性能通常与振膜刚性正相关，除此之外，这枚扬声器还使用CCAW线圈，CCAW其实就是铜包铝，铜包铝比纯铜线圈轻，这样可以提升响应速度，使用CCAW线圈的设计很常见。Buds 3 Pro 还经过“HRTF专业调音”，“录音室级校准”，“专业音乐工作者的挑剔耳朵，也能被满足”，这类词汇出现在小米文案中早就让人毫无波澜，但HRTF这个词汇还是唤醒了一点点沉睡的记忆，上次听到这个词差不多在20年前了。HRTF是Head Related Transfer Function缩写，中文意思为头部相关传递函数，它可以用于生成3D音效的听感，也可以用于声音的矫正，它可以用于降低环境导致的声音反射、混合、延时造成的负面影响，部分监听音箱有类似的环境自适应技术。但HRTF如何运用于耳机的调音，说实话，是我们孤陋寡闻了。

支持LHDC 4.0，也是Buds 3 Pro“HiFi级音质”的底气所在，我们目前并不清楚LHDC 4.0相比之前版本有了什么改进，单说音质，LHDC的表现是可以的，LHDC面临最大的问题其实是普及不足，LHDC小米新机型以及部分OPPO机型以白名单的方式支持，远不及LDAC的普及率，而最新的LHDC 4.0，至截稿日止，仅仅小米Civi一款手机支持。没有支持LHDC 4.0的手机，能不能以LHDC的旧版连接呢？答案是不行，即便是当前的在售旗舰小米11 ULTRA，也只能以SBC/AAC来连接Buds 3 Pro。花了大篇幅的大卖点，兼容性却非常糟糕，所以，LHDC 4.0到底是用来做宣传用的，还是给消费者用的呢？

 Buds 3 Pro 还是“小米首款空间音频” 设备，本质上是通过陀螺仪模拟固定方向的场景，用HRTF处理并用双声道呈现，目前仅仅支持小米Civi，并且需要爱奇艺的5.1片源配合，对于大部分用户，这个功能目前也没什么价值。

 Buds 3 Pro 造型很扁平，方便携带，可以方便的揣入口袋，或者背包的某个小收纳空间内，总重量48克。耳机含套重约5克，两只10克，标配大中小三组耳套，佩戴比较舒适，不容易脱落，但密封的稳定性存在一点问题，容易出现漏音现象，从而影响音质和主动降噪性能。

Buds 3 Pro在关闭主动降噪时，单次续航6小时，搭配电池盒总续航为27小时，在开启主动降噪时，单次续航则为5小时。实测表现与官方数据基本吻合。支持QI无线充电。

主动降噪和语音降噪
2021年，SOOMAL尝试使用新的测试方法来改进部分测试项目，使测试结果更易于呈现。我们来看看Buds 3 Pro在主动降噪方面的表现。
注意：测试所得数据不适合与其他平台的对比，因为测试环境、设备都有差别。

![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机-不同模式的噪声对比](https://images.soomal.cc/images/doc/20211226/00096869.webp)




我们通过模拟不同环境的噪声进行评估，最适合呈现的噪声就是白噪声。我们制造了一个80dB左右的白噪声环境，分别录制了环境噪声、三种主动降噪模式以及通透模式下的噪声。
![SOOMAL 降噪数据表[2021年12月30日]](https://images.soomal.cc/images/doc/20211230/00096934.webp)




如果以20-2000Hz频率段的整体表现为排序依据，会发现深度级的主动降噪表现是最差的，而轻度级反而是最好的，但如果以20-1000Hz为排序依据，那结果就又反过来了。
![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机-降噪曲线](https://images.soomal.cc/images/doc/20211226/00096870.webp)




这张图表中的红线代表噪声源的曲线，其他颜色代表Buds 3 Pro不同主动模式下的噪声曲线，一个格子的高度代表5dB。可以看到，500Hz似乎是个支撑点，增强了500Hz以下噪声的滤除，500Hz以上的噪声就会增大，就像个跷跷板。出现这种情况，说明降噪频率范围宽度存在不足，这也就出现了《降噪数据表》那种排序情况。SOOMAL测评的这套Buds 3 Pro经过一轮换新，上一套Buds 3 Pro在开启降噪后能听到明显的噪音，换新后问题消失。

![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机-语音降噪](https://images.soomal.cc/images/doc/20211226/00096871.webp)




通话降噪我们模拟了40dB的室内、80dB的白噪声、75dB左右的人声嘈杂、车流噪声的公共场所等场景，以及实景测试。Buds 3 Pro 对突如其来的噪声反应速度较快，大约0.5秒，对恒定噪声的消除效果优秀，语音保持清晰，但大力度的消除背景噪声也会导致主语音存在感冒声问题，但对语义理解不构成负面影响。Buds 3 Pro的通话语音降噪表现在当前属于一流水平。

音质与总结
![xiaomi 小米 Buds 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20211222/00096839.webp)




 Buds 3 Pro 虽然支持LHDC 4.0，但目前只有配合小米Civi手机才能满足使用条件，但我们手头并没有Civi，我们也没打算去为了Buds 3 Pro再去买一台Civi。测试中只好选择小米11 ULTRA作为音源设备，以AAC连接。

高频表现：Buds 3 Pro的高频延展并不算好，但听起来不会像FlipBuds Pro那么又糊又黯。其高频声线还算清晰，但不够细腻，大音量时颗粒感和毛刺感会变得明显，动态和瞬态表现一般，密度偏弱，在支持AAC的机型中，表现其实算比较好的一波了。如果使用LHDC，应该会有比较明显的改善。

中频表现：Buds 3 Pro中频风格柔和，结像居中，但结像点偏远偏虚，密度存在不足，也导致形态欠佳，声音也缺乏饱满感，懒散乏力，通俗的描述就像没吃饱饭。齿音现象少，听感偏暖。在爆棚音乐中，中频会受到全局影响。

低频表现：Buds 3 Pro低频风格是比较松，松不是坏事，但松松垮垮就不是好事了，其低频力度不足，密度也不够，瞬态也会受到影响，表现暴躁的重金属乐时尤为明显。表现中低速打击乐时，耐听度还可以。量感上比FlipBuds Pro收敛了不少。

 在扬声器素质上，Buds 3 Pro应该是比FlipBuds Pro的好，在调音上，Buds 3 Pro也肯定比FlipBuds Pro是进步了，但整体的听感上仍然有松垮乏力的问题，形态欠佳。使用Civi搭配的话，音质会有改善，但风格问题不会有什么变化。

总结：Buds 3 Pro 不管在造型、做工还是降噪、音质方面的表现，都是普普通通，就是一款平庸的产品。
