红米AirDots 3出了Pro版，这也是红米品牌下的首款主动降噪的真无线耳机，定位入门级，实际售价低于300人民币。之前SOOMAL测评了不少同级产品，表现都不太理想，AirDots 3 Pro会不会带来惊喜呢？为了行文方便，下文中的AirDots 3 Pro简写为ADP。

软硬件设计
![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095326_01.webp)




![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095327_01.webp)




ADP有黑灰两种配色，SOOMAL手头的这款是灰色款，采用磨砂质感的表面处理工艺，手感细腻，显得挺高级，抗油污能力一般，好像也不是特别耐划。其基础造型是一个跑道形，整体设计看上去圆溜溜的，设计风格上与AirDots 3有明显区别。模具精度优良，在同价位产品中绝对称得上一流，放在更高级别的产品中也算得上精致。盒盖铰链阻尼偏小但均匀，没有明显的松动感。
![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095330_01.webp)




![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095331_01.webp)




![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095332_01.webp)




![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095328_01.webp)




耳机采用不带把的豆型入耳式设计，采用与电池盒一样的表面处理工艺，触感温和，耳壳外侧突出跑道形线条，与电池盒基础造型相呼应，配有浅蓝色的仿宝石质感的装饰片，让设计变得生动起来，这种浅蓝灰的配色显得简洁干净。单只耳机重量4.9克，并不算重，但耳机突出耳道之外的部分较多，可能出现重心靠外的问题。佩戴舒适度一般，有一定的顶胀感，久戴会有轻度的不适。
![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-扬声器结构图[来自官网]](https://images.soomal.cc/images/doc/20210810/00095472.webp)




ADP 采用一枚9毫米口径的扬声器，从官方文案和结构图看，没有什么特殊设计，其实，对于目前大部分TWS耳机来说，扬声器尚构不成音质的瓶颈。

小米没有公布ADP的芯片信息，根据网上的一些拆解信息，可以了解到它采用的是络达 AIROHA AB1565AM，它支持蓝牙5.2。编解码支持方面，ADP仅支持SBC和AAC，不清楚是否芯片组本身的限制还是其他原因。

ADP 单只耳机内置电池容量为35毫安时，关闭降噪的单次续航时间为6小时，实测其实可达7小时，耳机支持盒内快充。充电盒内置电池470毫安时，关闭降噪的状态下，可以提供22小时的续航时间，总续航则为28小时。除了支持有线充电外，还支持QI无线充电，在这个价位的同类产品中不多见。
![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-连接小米手机-弹窗](https://images.soomal.cc/images/doc/20210810/00095474_01.webp)




![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-设置](https://images.soomal.cc/images/doc/20210810/00095475_01.webp)




![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-更多设置](https://images.soomal.cc/images/doc/20210810/00095476_01.webp)




较新的小米和红米机型在更新组件或者固件吼能提供ADP的原厂支持，第三方Android手机可以通过《小爱同学》来实现全功能支持，与小米集成的一样。《小爱同学》的也有iOS版，最新版支持ADP。虽然耳机定位不高，但功能支持方面并没有偷懒，也没有强行制造品牌原生优势，对第三方平台一视同仁，立场开放。

主动降噪和语音降噪
2021年，SOOMAL尝试使用新的测试方法来改进部分测试项目，使测试结果更易于呈现。我们来看看ADP在主动降噪方面的表现。
注意：测试所得数据不适合与其他平台的对比，因为测试环境、设备都有差别。

![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-不同模式的噪声对比](https://images.soomal.cc/images/doc/20210810/00095479.webp)




我们通过模拟不同环境的噪声进行评估，最适合呈现的噪声就是白噪声。我们制造了一个80dB左右的白噪声环境，分别录制了环境噪声以及被动降噪[降噪关闭]、降噪[深度]、降噪[均衡]、降噪[轻度]、通透、人声增强模式下的噪声。
![SOOMAL 降噪数据表[2021年8月10日]](https://images.soomal.cc/images/doc/20210810/00095482.webp)




不管是那种降噪模式，ADP的表现都很不理想，当前末流水准。降噪[深度]的降噪重点在500Hz一下频率，实听是有效果的，毕竟它能过滤掉这个频段一半的噪声，但和优秀机型没法比。ADP的问题不仅仅是降噪深度不够，宽度也不够。

![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-降噪曲线](https://images.soomal.cc/images/doc/20210810/00095480.webp)




这张图表中的红线代表噪声源的曲线，其他颜色代表了ADP不同模式下的噪声曲线，一个格子的高度代表5dB。ADP任何一条曲线都达不到理想水平，真的是很入门的级别。
![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机-语音降噪](https://images.soomal.cc/images/doc/20210810/00095481.webp)




通话降噪我们模拟了40dB的室内、80dB的白噪声、75dB左右的人声嘈杂、车流噪声的公共场所等场景，以及实景测试。ADP对于对于恒定噪声有着优良的消除效果。降噪对人声有一定的负面影响，人声可能出现沙沙声和感冒声，但结合语境，应该不会造成语义理解歧义。实测右耳摘机通话有点小效果，但不明显，实用性不足。

音质与总结
![红米 Redmi AirDots 3 Pro 蓝牙真无线主动降噪耳机](https://images.soomal.cc/images/doc/20210729/00095329.webp)




ADP 仅支持SBC和AAC，当前主流手机机型都支持这两种编解码方案，但我们还是选择了小米11 Ultra手机作为数字音源，毕竟是同一家公司旗下的产品。

高频表现：ADP的高频明显偏暗，很多高频乐器的泛音区难以呈现细节，高频大概在8kHz以上频率段会出现明显的衰减。ADP以AAC作为默认解码方案，和小米11 Ultra配合，会丢失16kHz以上的信号。

中频表现：ADP的中频表现良好，听上去比小米 FlipBuds Pro更好，听感更为扎实饱满，没那么松松垮垮。ADP的没有去做刻意的修饰和拔高，也没有明显的齿音。细节表现偏弱，结像力普普通通，不好也不坏。

低频表现：ADP的中低频衔接明显比FlipBuds Pro更好，没有明显的凹陷感，低频部分还是有修饰痕迹，有增强处理，但还算合理，低频有量感，力度偏软，牺牲了一点瞬态和动态以及解析力，算是找到了一个恰当的平衡点。

ADP的声音表现还是呈现一种黯淡拘谨的听感，听起来有点闷，细节有欠缺，但还属于正常的声音，没有那种搞怪的调音，耐听度也还可以，放在整个TWS耳机中对比，谈不上多优秀，看看价格，我们觉得还行。

总结：ADP作为红米的首款主动降噪耳机，在主动降噪方面表现并不理想，主动降噪无法成为真正的吸引点。ADP的优势在于做工，300上下的产品做到这种精致程度，并不多见，软件的配合也做得到位。音质方面虽然有明显的不足，但也并不是不可接受，毕竟ADP的价格还是比较实惠的。
